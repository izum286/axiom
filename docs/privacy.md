# Privacy Policy

Axiom is an open-source plugin that runs entirely within your local AI coding assistant. It does not collect, transmit, or store any user data.

## What Axiom Does

Axiom provides skills (markdown instruction files) that your AI coding assistant reads locally. All processing happens on your machine within your coding tool's existing session.

## What Axiom Does NOT Do

- Collect personal information
- Send data to external servers
- Track usage or analytics
- Access files outside your coding session
- Store any data between sessions

## Third-Party Services

Axiom's MCP server, if used, runs locally and does not connect to external services.
